# Sales & Revenue Analysis Dashboard

## Objective

The main objective of this project is to analyze sales and revenue data using Python and Power BI and create an interactive dashboard for business insights.

This project helps in understanding:

- Total Sales
- Total Profit
- Monthly Sales Trends
- Region-wise Sales Performance
- Top Performing Products
- Category-wise Profit Analysis
- 
## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Power BI
- Excel
- VS Code

## Dataset Used

Dataset Name: sales.csv

The dataset contains:

- Order Date
- Sales
- Profit
- Product Name
- Region
- Category
- Customer Name

This dataset is used to perform sales analysis and create dashboard visualizations.


## Steps Performed

### 1. Data Collection

Imported sales dataset from CSV file.

### 2. Data Cleaning

- Removed duplicate rows
- Filled missing values
- Converted date columns
- Standardized column names

### 3. KPI Calculations

Calculated:

- Total Sales
- Total Profit
- Average Revenue
- Monthly Sales Performance

### 4. Data Visualization

Created charts for:

- Monthly Sales Trend
- Region-wise Sales
- Category-wise Profit
- Top Products Analysis

### 5. Power BI Dashboard

Built an interactive dashboard using:

- KPI Cards
- Charts
- Filters
- Slicers

## Dashboard Features

### KPI Cards

- Total Sales
- Total Profit

### Charts

- Monthly Sales Trend
- Region-wise Sales
- Category-wise Profit
- Top 10 Products

### Filters

- Region
- Category
- Year

## Output

Final output of this project includes:

- Cleaned sales dataset
- Python analysis code
- Power BI dashboard
- Dashboard screenshots
- Sales insights and business analysis

## Conclusion

This project helps businesses understand their sales performance and revenue trends using data visualization and dashboard reporting.

It improves decision-making by identifying profitable regions, top products, and sales growth opportunities.
